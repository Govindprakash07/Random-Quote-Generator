# Random Quote Generator 🌟  

A simple JavaScript app that displays a random motivational quote.  

## Features  
✔️ Generates random quotes  
✔️ Simple and responsive UI  

## Technologies Used  
- HTML  
- CSS  
- JavaScript  

## How to Run  
1. Download the project.  
2. Open `index.html` in a browser.
